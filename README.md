# Back-end-Coding-Challenge
API JSON built in Rust periodically consults the Mempool lightning node state, process the data, and stores it in a PostgresSQL database. The API features a GET method to retrieve the node's data.

## Build tools & versions used
- Axum 
- Diesel 
- PostgresSql (Supabase)

Inspired by https://youtu.be/v0axYVJX_hI
